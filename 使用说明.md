## 💾 如何使用真实数据

### 当前状态
⚠️ 目前使用模拟数据（用于演示）

### 切换到真实数据

1. **在项目目录创建data文件夹**
   ```bash
   cd /Users/ivan/Desktop/mh_w
   mkdir data
   ```

2. **复制MediaCrawler的数据文件到data目录**
   - CSV文件：`data/*.csv`
   - JSON文件：`data/*.json`  
   - SQLite数据库：`data/*.db`

3. **推送更新到GitHub**
   ```bash
   git add .
   git commit -m "添加真实数据"
   git push
   ```

4. **等待Vercel自动部署**
   - 推送后约30秒-1分钟
   - Vercel会自动检测并重新部署
   - 无需手动操作
